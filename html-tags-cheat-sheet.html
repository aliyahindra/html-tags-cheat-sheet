Headings

Headings from 'h1' to 'h6' indetify blocks and sections. Every page must have one 'h1'.

Paragraphs

Paragraphs of text go in 'p' tags.

Lists

Unordered Lists

Unordered list are marked with bullet points and go in 'ul' tags. Each list item goes in an 'li' tag.

Types of Lists:

  • Unordered lists ('ul')
  • Ordered lists ('ol')

Ordered Lists

Ordered lists are numbered and go in 'ol' tags. Each list item goes in an 'li' tag.

Anchors (i.e. links)

Anchor tags create links to other pages. The URL goes in the 'href' attribute. Go to DuckDuckGo to search for more information.

Settin 'target="blank"' will open the link in a new window. See MDN for more details.

Anchor tags can also creat links to a new location on the same page. Instead of putting a URL goes in the "href" attribute, just use the "id" of the element you want to navigate to. This allows you to Jump to the top of the page.

Text Emphasis

Italics

The 'em' element adds extra emphasis to the text within the text within the opening and closing tags. This shows up as italic text in most browsers.

Bold

You can also use the 'strong' element to emphasize text. This shows up as bold text in most browsers.

Organizing Content with 'div' and 'span'

'div'

The 'div' element is a block element, so each 'div' will show up on a new line.

'span'

The 'span' element is an inline element, so there will be no line break between each 'span'.
